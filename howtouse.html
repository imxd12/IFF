üí∞ Spendly - Income & Expense Tracker

Track daily income and expenses with detailed categories. View balance and export reports.

How to Use Spendly:

  • Adding Income: Click the "Income" tab, select date, amount, category (Salary, Business, Gift, etc.), add notes, and save
  • Adding Expense: Click the "Expense" tab, choose from 11 categories (Food, Travel, Shopping, Bills, etc.), select subcategory, enter amount and save
  • View Transactions: All entries appear in the list below with type, date, category, and amount
  • Edit/Delete: Click any transaction to edit details or delete it
  • Quick Add: Use the floating action button (FAB) in bottom-right for quick expense entry
  • Filter Data: Use date picker or search box to find specific transactions
  • Export Data: Click "Export CSV" to download your data for backup or analysis
  • Import Data: Use "Import JSON" to restore previously backed up data

Available Categories:

Food, Travel, Rent, Shopping, Bills, Health, Entertainment, Education, Savings, Family, Other - each with multiple subcategories

üíº Nexus - Work & Earnings Tracker

Log work items, hours, bonuses, and penalties. Perfect for gig workers and freelancers.

How to Use Nexus:

  • Create Entry: Click "Add New Entry" button or use FAB
  • Fill Details: Enter date, select work category (Freelance, Delivery, Design, Development, etc.)
  • Track Metrics: Input items completed, hours worked, bonus earned, and any penalties
  • Auto Calculate: Payout is calculated automatically as: (Items √ó 10) + (Hours √ó 50) + Bonus - Penalty
  • Manual Override: You can manually enter payout amount if calculation doesn't fit your work
  • Add Notes: Describe the work done for future reference
  • View Analytics: Check daily earnings trend, category distribution, items vs hours comparison, and bonus/penalty analysis
  • Export Reports: Download JSON or CSV files for record keeping

Analytics Features:

  • Daily Earnings Trend Chart (Last 14 Days)
  • Category-wise Earnings Distribution
  • Items vs Hours Comparison
  • Bonus vs Penalty Analysis
  • Total Items, Hours, Bonus, Penalty metrics

üìÖ PocketCal - Calendar Money Tracker

Calendar view of pocket money received. See monthly totals and daily averages.

How to Use PocketCal:

  • Navigate Calendar: Use Previous/Next buttons to switch months
  • Add Entry: Click any date on calendar to add/edit entry for that day
  • Quick Add Today: Use "Add Today" button or FAB to quickly log today's amount
  • Jump to Date: Use date picker to navigate to specific date quickly
  • View at Glance: Days with entries show green highlight and amount below date
  • Today's Date: Current date highlighted with blue border
  • Categories: Choose from Pocket Money, Gift, Allowance, Reward, Work, Education, Festival, Other
  • View Charts: Analyze daily amounts, category distribution, weekly comparison, and 6-month trend

Summary Statistics:

  • This Month Total - Total amount received this month
  • All-Time Total - Cumulative amount since tracking started
  • Highest Day - Maximum amount received in a single day
  • Average/Day - Average daily amount for entries logged
  • Days Tracked - Total number of days with entries
  • This Week Total - Amount received in current week
  • Best Month - Highest monthly total ever recorded

üí° Essential Tips & Best Practices

  • Regular Backups: Export your data weekly using "Export" buttons in each module
  • Consistent Categorization: Use categories consistently for accurate analytics
  • Daily Updates: Update entries daily to maintain accurate financial picture

üîß Troubleshooting

  • Data Not Saving? Check browser settings - ensure localStorage is enabled
  • Charts Not Loading? Refresh page or check internet connection (Chart.js requires CDN)
  • Lost Data? Always keep export backups - data clears if browser cache is cleared
  • Modal Not Closing? Click outside modal area or use X button
  • Theme Not Switching? Clear cache and reload page
