How to Use Spendly

  • Adding Income: Click the "Income" tab, select date, enter amount, choose category (Salary, Business, Gift, Investment, etc.), add optional notes, and click save
  • Adding Expense: Click the "Expense" tab, choose from 11 categories (Food, Travel, Shopping, Bills, Health, Entertainment, Education, etc.), select relevant subcategory, enter amount and description, then save
  • View Transactions: All entries appear in chronological list below with transaction type indicator, date, category icon, description, and amount
  • Edit Entry: Click any transaction card to open edit modal, modify details, and save changes
  • Delete Entry: Open transaction details and click delete button (confirmation required)
  • Quick Add: Use the floating green + button in bottom-right corner for rapid expense entry without opening full form
  • Filter by Date: Use date range picker to view transactions from specific time periods
  • Search Transactions: Type in search box to filter by category, description, or amount
  • Export Data: Click "Export CSV" button to download spreadsheet for Excel/Google Sheets analysis
  • Import Backup: Use "Import JSON" to restore previously exported backup data
