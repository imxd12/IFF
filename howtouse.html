How to Use

Mastering MoneyFlow

A fluid guide to the Antigravity Financial Engine.

1. The Hub (Dashboard)

The Hub is your liquid financial overview. It continuously calculates your spending velocity and provides Smart Insights. Watch out for AI voice greetings, and if you haven't already, install MoneyFlow as a PWA for a seamless full-screen experience!

2. Spendly Analytics

Log your daily income and expenses here. Use the toggle buttons to switch forms. Every entry updates the dynamic charts instantly. Need to present a report? Use the Export PDF button to generate a sleek Bank Passbook.

3. PocketCal Engine

PocketCal is a visualized day-grid for your Pocket Money tracking.

Magic Sync: When you add an income labeled "💵 Pocket Money" in Spendly, it automatically populates the Calendar here, keeping your total balance unified across engines!

4. The Inky Theme Switch

Tap the Sun/Moon icon in the header to activate the Inky Transition Layer. This utilizes advanced ViewTransitions API with a clip-path circle expansion, swapping the entire UI between Ceramic White and Obsidian Black flawlessly.

5. Bank Passbook Exports

Need official reports? In Spendly or PocketCal, use the Export PDF button to generate standardized Bank Passbooks. Choose from Month or All Time filters. The PDF retains professional Debit (Dr) / Credit (Cr) structures, dynamically calculated Running Balances, and pristine visual styling using purely client-side compilation.

6. The Antigravity Ecosystem

MoneyFlow utilizes a cutting-edge Antigravity Physics Engine. As you traverse the interface, observe how distinct cards and data elements float upward at unalike velocities. This GSAP ScrollTrigger-powered parallax, combined with Lenis smooth scrolling, builds a truly weightless user experience.
